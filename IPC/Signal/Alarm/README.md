### Simple example of Alarm signal generation and handling.

### Compilation Steps
- To generate executable use command:
``` 
    make all
```
- To remove output files use command:
```
    make clean
```

### Running the example
- Run the example as follow
```
    $ ./alarm
    Remaining time_1 0 
    Remaining time_2 3 
    Alarm signal handler invoked
```
